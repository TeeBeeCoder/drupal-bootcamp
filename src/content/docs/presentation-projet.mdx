Dans ce bootcamp, nous allons réaliser un site e-commerce complet basé sur le template **Tailstore**.

## Le projet Tailstore

Tailstore est un template e-commerce moderne et responsive construit avec **Tailwind CSS**. Il offre une base solide pour créer une boutique en ligne élégante et fonctionnelle.

Vous pouvez retrouver le code source original du template sur GitHub : [spacemadev/Tailstore](https://github.com/spacemadev/Tailstore).

## Fonctionnalités principales

Le projet comprendra les fonctionnalités suivantes :

- **Page d'accueil** :
  - Slider de mise en avant (Swiper Slider)
  - Carrousel des marques partenaires
  - Sections "Derniers produits" et "Produits populaires"
  - Section Blog
  - Formulaire d'inscription à la newsletter

- **Boutique (Shop)** :
  - Filtres de produits (catégories, prix, etc.)
  - Liste des produits avec grille responsive
  - Pagination

- **Fiche produit** :
  - Galerie d'images
  - Description détaillée
  - Avis clients
  - Produits similaires

- **Fonctionnalités e-commerce** :
  - Panier d'achat
  - Tunnel de commande (Checkout)
  - Compte client (Connexion / Inscription)

Ce projet nous servira de fil rouge pour explorer toutes les facettes de Drupal, de la modélisation des contenus à la création de thèmes personnalisés, en passant par la configuration de modules et le développement sur mesure.
